# 光线传感器例程

### 1 简介

本例程为光线传感器的控制例程，对应目录如下：

- ProjectLight
  - src
    - lib
      - Demo

### 2 要点

- I2C相关接口的常规用法
- OPT3001光线传感器库函数的常规用法

### 3 实验条件

硬件：Lierda NB86 EVK/NB86-G模组  
软件：USB转串口驱动  
集成开发环境：Eclipse
